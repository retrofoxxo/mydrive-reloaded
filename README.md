# mydrive-reloaded
Revives the MyDrive Mobile app discontinued by TomTom.

Mainly revived the app, so I could use smartphone notifications again lol.

What works so far
- Login
- Sign Up
- Map screen
- Pairing TomTom
- Setting up smartphone notifications
- Traffic (provided by tomtom may stop working)
- Searching Places (provided by tomtom may stop working)
- Traffic and other services via MyDrive Mobile app (not Bluetooth tethering)

What does not work
- My Places
- Directions
- Sending route to device
- Viewing subscription info
- My Routes
- My Trips
- Anything not listed above in the what works so far tab
